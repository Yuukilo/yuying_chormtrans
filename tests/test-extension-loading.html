🌐 语影翻译扩展加载测试

📋 测试步骤

  1. 确保已经构建了扩展程序 (运行 npm run build)
  2. 打开Chrome浏览器，进入扩展程序管理页面 (chrome://extensions/)
  3. 开启"开发者模式"
  4. 点击"加载已解压的扩展程序"
  5. 选择项目的 dist 文件夹
  6. 检查扩展是否成功加载，没有错误信息
  7. 在此页面测试翻译功能
